Smart Money Monitor

A Python-based automation system that monitors NSE bulk deals and detects trades executed by high-frequency trading firms. The system fetches market data, filters targeted firms, generates reports, and sends automated email alerts.

Problem

Raw exchange data contains valuable signals about institutional and high-frequency trading activity. However, monitoring NSE Bulk Deals manually every day is inefficient and time-consuming. Traders often miss important moves from major algorithmic trading firms such as HRTI or Graviton.

Existing platforms also limit the number of investors or entities you can track, making it difficult to follow multiple institutional players simultaneously.

Solution

This project builds an automated monitoring system that fetches NSE bulk deal data, filters trades executed by selected high-frequency trading firms, and generates daily reports.

The system runs automatically through scheduled workflows, exports structured CSV data, and sends email alerts whenever relevant trades are detected.

Tech Stack

  • - Python
  • - Requests (API data fetching)
  • - Pandas (data processing)
  • - SMTP Email Automation
  • - GitHub Actions (scheduled automation)
  • - CSV Reporting

Key Features

  • - Automated NSE bulk deal monitoring
  • - Detection of selected HFT firms
  • - CSV report generation
  • - Automated email alerts
  • - Scheduled execution via GitHub Actions
