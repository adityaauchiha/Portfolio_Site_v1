AI Telegram Diet Coach

A multi-user AI powered Telegram bot that helps users track weight progress, set fitness goals, and receive AI-based progress insights.

Problem

Many people struggle to stay consistent with their fitness goals. Tracking weight manually and understanding progress trends can be difficult without structured guidance or feedback.

Solution

This Telegram bot allows users to track their weight journey, set targets, and receive AI-generated progress analysis. The system stores user data securely and provides motivational feedback based on individual progress.

Tech Stack

  • Python
  • python-telegram-bot (Async)
  • SQLite Database
  • OpenAI API Integration
  • Telegram Bot API

- Features include multi-user support, AI progress analysis, rate-limited AI usage, goal management, and restartable fitness journeys.
